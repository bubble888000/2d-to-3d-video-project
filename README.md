# 2d-to-3d-video-project
# 2D转3D视频项目（含效果视频）说明文档

## 项目简介
基于开源项目 `nunif` 实现 2D 视频到立体 3D 视频的转换，使用 PyTorch 完成模型推理与流程搭建，实现视频深度感知与左右眼分屏效果。

## 工作内容
- 完成环境配置、模型加载与推理脚本运行
- 处理 **30+ 段** 视频，优化推理参数，提升处理效率 **15%**
- 输出可播放的 3D 立体视频结果

## 技术栈
- Python
- PyTorch
- 视频处理
- 深度学习推理
- 数据流程搭建
